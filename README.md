## ✔️Installation
- git clone https://edward-tobilko.github.io/Weather-App/
- `npm install`
- `npm run start`
___
### Tech stack:
- React (Hooks)
- ANTD / Normalize / ChartJS / Lodash / Luxon / Moment
- API: "https://api.openweathermap.org/" / Axios
- Git / GitHub
